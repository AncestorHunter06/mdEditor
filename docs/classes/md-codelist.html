md-codelist Class

Module: components-input
Parent Module: mdeditor
Defined In: app/pods/components/input/md-codelist/component.js:7

Specialized select list control for displaying and selecting options in mdCodes codelists. Access to codelists is provided by the 'codelist' service. Descriptions of all codes (tooltips) are embedded within the codelists.
